## 一、产品概述

### 1.1 产品定位

基于网页端的城市活动用户平台，整合品牌展示、活动运营、用户参与和后台管理于一体。核心活动为每周举办的城市剧本杀类线下活动，通过线上全流程闭环（宣传→报名→注册→邀请→参与→任务→通关→成就）驱动用户持续参与。

### 1.2 目标用户

| 角色 | 描述 |
|------|------|
| **普通访客** | 未注册的网站浏览者，通过广告或链接进入 |
| **注册用户** | 已注册并购买过活动的参与者 |
| **活动参与者** | 正在参与当期活动的用户 |
| **管理员** | 系统运营方，负责活动配置、用户管理、邀请函发放等 |

### 1.3 核心价值

- 为用户提供沉浸式城市活动体验
- 为品牌合作提供展示与联动入口
- 为运营方提供完整的活动管理工具
- 通过电子印章体系激励用户持续参与

## 三、功能模块详细描述

### 3.1 品牌展示模块

#### 3.1.1 品牌官网

**页面描述：** 作为品牌的主门户页面，展示品牌故事、核心理念、活动概览及最新动态。

| 功能项 | 描述 |
|--------|------|
| 品牌介绍 | 品牌故事、创始理念、团队风采的图文展示 |
| 活动预告 | 展示即将举办的活动，引导用户报名 |
| 动态资讯 | 品牌相关新闻、活动回顾、媒体报道 |
| 导航入口 | 链接到联名合作页、活动报名页、用户中心等 |
| 底部信息 | 联系方式、社交媒体链接、隐私政策 |

#### 3.1.2 联名合作品牌官网

**页面描述：** 展示与联名合作品牌的联合活动、合作故事及双方品牌价值。

| 功能项 | 描述 |
|--------|------|
| 合作品牌展示 | 联名品牌 Logo、介绍、合作背景 |
| 联名活动专区 | 专属联名活动的宣传与报名入口 |
| 合作故事 | 双方合作的起源、理念契合点 |
| 产品/服务展示 | 联名限定产品或服务介绍 |

---

### 3.2 活动运营模块

#### 3.2.1 活动宣传与报名页

**页面描述：** 每期活动的专属宣传页面，承载广告投放落地和用户报名转化的核心链路。

| 功能项 | 描述 |
|--------|------|
| 活动主题展示 | 当期活动的主题名称、视觉海报、主题介绍 |
| 活动详情 | 时间、地点、参与人数、活动流程概览 |
| 费用说明 | 活动价格、包含内容、优惠政策 |
| 往期回顾 | 历史活动精彩片段（图文/短视频） |
| 报名入口 | 「立即报名」按钮，引导未注册用户先注册，已注册用户直接购买 |
| 常见问题 | 活动相关 FAQ |

**关键流程：**

```
广告/分享链接 → 活动宣传页 → [未注册] → 注册/登录 → 购买 → 报名成功
                           → [已注册] → 购买 → 报名成功
```

#### 3.2.2 活动邀请函页面

**页面描述：** 由管理员为每期活动统一配置生成的专属邀请函，通过独立链接私发给已报名用户。每份邀请函具有唯一性。

| 功能项 | 描述 |
|--------|------|
| 邀请函展示 | 精美的邀请函视觉呈现，包含活动主题元素 |
| 活动信息 | 本期活动的具体时间、集合地点、注意事项 |
| 专属信息 | 用户姓名（或昵称）、专属编号 |
| 确认参与 | 用户点击确认参与，系统记录出席状态 |
| 有效期提示 | 邀请函的有效时间范围 |

**访问规则：**
- 仅通过管理员私发的专属链接访问
- 链接与用户账号绑定，不可转让
- 过期链接自动失效

---

### 3.3 用户中心模块

#### 3.3.1 用户注册与登录

**页面描述：** 用户账号的创建与认证入口。

| 功能项 | 描述 |
|--------|------|
| 注册 | 手机号/邮箱注册，设置密码，填写基础信息（昵称、头像） |
| 登录 | 账号密码登录 |
| 忘记密码 | 通过手机/邮箱验证重置密码 |
| 用户协议 | 注册时需同意的用户服务协议 |

#### 3.3.2 个人中心首页

**页面描述：** 用户登录后的主控面板，汇总展示个人信息、活动状态和成就。

| 功能项 | 描述 |
|--------|------|
| 个人信息卡片 | 头像、昵称、注册时间、参与次数统计 |
| 我的活动 | 当前参与的活动状态（待开始/进行中/已完成） |
| 最近邀请函 | 最新一期邀请函的快捷入口 |
| 电子印章墙 | 已获得的电子印章展示（详见 3.3.5） |
| 快捷导航 | 到各功能页面的入口 |

#### 3.3.3 个人信息页

**页面描述：** 用户查看和编辑个人资料的页面。

| 功能项 | 描述 |
|--------|------|
| 基本信息 | 头像、昵称、真实姓名、联系方式、邮箱 |
| 修改密码 | 修改登录密码 |
| 活动历史 | 参与过的所有活动列表，包含每期主题和参与日期 |
| 电子印章 | 每次参与完成后获得的电子印章，按时间排列 |

#### 3.3.4 活动信息与任务页

**页面描述：** 用户在活动期间查看当期活动详情和参与线上任务的核心页面。

| 功能项 | 描述 |
|--------|------|
| 当期活动信息 | 活动主题、地图/路线、NPC 点位提示、注意事项 |
| 活动进度 | 当前活动阶段的可视化展示 |
| 线上任务卡入口 | 进入线上任务的入口按钮 |
| 实时提示 | 活动过程中的重要通知和提示 |

#### 3.3.5 线上任务卡页面

**页面描述：** 活动中的核心互动页面，用户在此接收任务、输入密码、推进通关流程。

| 功能项 | 描述 |
|--------|------|
| 任务发放 | 展示当前需要完成的任务描述和线索 |
| 密码输入框 | 用户从 NPC 处获得密码后在此输入验证 |
| 任务进度 | 已完成/未完成任务的进度条展示 |
| 通关状态 | 全部任务完成后的通关祝贺页面 |
| 电子印章发放 | 通关后自动发放本期电子印章到个人中心 |

**关键流程：**

```
活动开始 → 查看任务卡 → 线下寻找 NPC → 获得密码 → 线上输入密码
→ 验证通过 → 解锁下一任务 → ... → 全部通关 → 获得电子印章
```

#### 3.3.6 电子印章系统

**页面描述：** 用户参与活动的成就体系，以电子印章为核心载体。

| 功能项 | 描述 |
|--------|------|
| 印章展示 | 以收藏墙/日历形式展示所有已获得印章 |
| 印章详情 | 每枚印章可点击查看对应活动的主题、日期、地点 |
| 印章样式 | 每期活动对应不同设计的专属印章 |
| 分享功能 | 可将印章墙生成图片分享到社交媒体 |

### 3.4 管理后台模块

#### 3.4.1 管理员登录

**页面描述：** 管理员专属的登录入口，与用户端分离。

| 功能项 | 描述 |
|--------|------|
| 账号密码登录 | 管理员账号认证 |
| 安全机制 | 登录失败限制、会话超时自动退出 |

#### 3.4.2 仪表盘首页

**页面描述：** 管理员登录后的概览页面，展示系统核心运营数据。

| 功能项 | 描述 |
|--------|------|
| 数据概览 | 总用户数、本周新增用户、本周活动参与人数 |
| 本周活动状态 | 当期活动的报名情况、邀请函发放情况 |
| 快捷操作 | 常用功能的快捷入口（创建活动、生成邀请函等） |
| 近期动态 | 最近的用户注册、报名、通关等事件流水 |

#### 3.4.3 活动管理

**页面描述：** 管理员创建、编辑和管理所有活动的页面。

| 功能项 | 描述 |
|--------|------|
| 活动列表 | 所有历史及未来活动的列表，支持按状态筛选 |
| 创建活动 | 填写活动主题、时间、地点、描述、费用、人数上限等 |
| 编辑活动 | 修改已有活动的各项信息 |
| 活动上下架 | 控制活动在用户端的可见性 |
| 报名管理 | 查看某期活动的报名用户列表、报名状态 |
| 数据统计 | 单期活动的报名数、参与数、通关率等 |

#### 3.4.4 邀请函管理

**页面描述：** 管理员配置、生成和发放邀请函的核心操作页面。

| 功能项 | 描述 |
|--------|------|
| 邀请函模板管理 | 创建和编辑邀请函的视觉模板（背景、字体、布局） |
| 生成邀请函 | 选择活动 → 选择用户（已报名）→ 批量生成专属邀请函链接 |
| 邀请函列表 | 查看所有已生成的邀请函，包含用户、活动、状态、链接 |
| 链接管理 | 复制链接、查看链接访问状态、作废链接 |
| 私发辅助 | 标记已发送/未发送状态，辅助管理员逐一私发 |

#### 3.4.5 用户管理

**页面描述：** 管理员查看和管理所有注册用户的页面。

| 功能项 | 描述 |
|--------|------|
| 用户列表 | 所有注册用户的列表，支持搜索、筛选（按注册时间、参与次数等） |
| 用户详情 | 查看某用户的完整信息：基本资料、参与历史、获得印章 |
| 用户状态管理 | 启用/禁用用户账号 |
| 导出功能 | 导出用户数据（用于运营分析） |

#### 3.4.6 任务配置

**页面描述：** 管理员为每期活动配置线上任务内容的页面。

| 功能项 | 描述 |
|--------|------|
| 任务列表 | 按活动查看和管理所有任务 |
| 创建任务 | 设置任务描述、线索内容、NPC 密码、任务顺序 |
| 编辑任务 | 修改已有任务的内容 |
| 任务关联 | 将任务绑定到具体活动 |
| 通关条件 | 设置完成所有任务后的通关条件和印章发放规则 |

#### 3.4.7 品牌与联名管理

**页面描述：** 管理品牌官网和联名合作官网内容的页面。

| 功能项 | 描述 |
|--------|------|
| 品牌内容管理 | 编辑品牌官网的各板块内容（图文） |
| 联名合作管理 | 创建和管理联名合作品牌信息 |
| 联名活动关联 | 将联名品牌与具体活动绑定 |
| 素材管理 | 上传和管理品牌 Logo、海报、图片等素材 |

#### 3.4.8 系统设置

**页面描述：** 系统级的配置管理页面。

| 功能项 | 描述 |
|--------|------|
| 管理员账号管理 | 添加/编辑管理员账号、分配权限 |
| 网站基础设置 | 网站名称、Logo、备案信息等 |
| 通知设置 | 配置系统通知的方式和模板 |
| 操作日志 | 管理员的操作记录审计 |

## 四、核心用户流程

### 4.1 新用户首次参与活动完整流程

```
[外部广告/分享]           [平台内]
     │
     ▼
 活动宣传页浏览
     │
     ▼
 点击「立即报名」──→ 未注册 ──→ 注册页面 ──→ 注册成功 ──┐
     │                                                    │
     └──→ 已注册 ──→ 登录 ──────────────────────────────┘
                                                        │
                                                        ▼
                                                   购买活动
                                                        │
                                                        ▼
                                                  报名成功确认
                                                        │
                                                        ▼
                                            管理员生成邀请函链接
                                                        │
                                                        ▼
                                            管理员私发邀请函给用户
                                                        │
                                                        ▼
                                               用户打开邀请函
                                                        │
                                                        ▼
                                                  确认参与
                                                        │
                                                        ▼
                                               线下活动当日
                                                        │
                                                        ▼
                                           用户进入活动信息/任务页
                                                        │
                                                        ▼
                                              查看任务卡 → 线下寻找NPC
                                                        │
                                                        ▼
                                                   获得密码
                                                        │
                                                        ▼
                                               线上输入密码验证
                                                        │
                                                        ▼
                                              逐个完成所有任务
                                                        │
                                                        ▼
                                                  通关成功
                                                        │
                                                        ▼
                                             获得本期电子印章
                                                        │
                                                        ▼
                                          印章展示在个人中心印章墙
```

### 4.2 老用户再次参与流程

```
收到新一期活动通知
     │
     ▼
 登录 ──→ 活动宣传页 ──→ 购买 ──→ 报名成功
                                      │
                                      ▼
                           管理员私发新邀请函
                                      │
                                      ▼
                           打开邀请函 → 确认参与
                                      │
                                      ▼
                           活动日 → 任务 → 通关 → 新印章
```

## 六、非功能性说明

| 维度 | 说明 |
|------|------|
| **访问方式** | 纯网页端，支持手机浏览器和桌面浏览器访问 |
| **活动频率** | 每周举办一次，长期持续运营 |
| **数据延续** | 用户数据、活动历史、电子印章永久保存 |
| **并发预期** | 单期活动参与者数十至数百人，非高并发场景 |
| **内容更新** | 品牌页面和活动页面内容需支持灵活更新 |
