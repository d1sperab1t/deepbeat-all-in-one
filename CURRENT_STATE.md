## 技术决策记录 📝

| 决策 | 选择 | 原因 |
|------|------|------|
| 数据库 | SQLite (sql.js WASM) | 轻量、无需原生编译、适合单机部署 |
| 前端框架 | 原生 JS + CSS | 简单直接、无构建工具依赖 |
| 路由 | 自定义 hash-based | 轻量级 SPA，足够需求 |
| 部署方式 | 腾讯云 Docker | 与现有 WordPress 共存 |
| 认证 | JWT（用户7天/管理员24h） | 无状态、易扩展 |
